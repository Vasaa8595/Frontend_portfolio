PCB Design

PCB Design is the process of creating a physical layout for electronic circuits. It involves placing components and connecting them with copper traces on a board using software tools like Altium ,Ki-Cad, Allegro, Eagle. It ensures the circuit works efficiently and fits into compact electronic devices.A well-designed PCB ensures efficient signal flow, compact size, reduced noise, and reliable performance in electronic devices.

Latest Projects

IOT - Trainer Kit

The IoT Trainer Kit is a compact, custom-designed 4-layer PCB that integrates multiple sensors and popular microcontrollers like Arduino Nano, ESP32, and Raspberry Pi. It is built to simplify IoT learning and experimentation through modular interfaces and real-time data interaction. The kit supports hands-on training in sensor integration, wireless communication, and embedded systems. Ideal for students, educators, and developers, it promotes innovation and rapid prototyping in IoT applications.

8 - Bit Computer

The 8-bit computer, inspired by Ben Eater’s design, is a modular, breadboard-based system that demonstrates the fundamentals of computer architecture. It covers key concepts like binary logic, clock cycles, memory addressing, and instruction execution. Built from basic logic ICs, the project offers a hands-on approach to understanding how computers work at the hardware level. It's an excellent educational tool for learning low-level computing and digital design.

Maze Solver

The Maze Solver project uses an STM32 microcontroller to autonomously navigate through a maze using sensor inputs and pathfinding algorithms. It features real-time decision-making, obstacle detection, and memory-based path optimization. The system demonstrates embedded control, efficient algorithm implementation, and autonomous robotics. It's ideal for showcasing practical applications of embedded systems in automation and AI-driven navigation.

RP2040-Based GSM Communication System

This project integrates the RP2040 microcontroller with a GSM module (SIM800/SIM900) to enable SMS-based communication for remote monitoring and control. It utilizes UART communication and AT commands to send and receive messages over cellular networks. The system is ideal for IoT applications requiring low-power, wireless connectivity in areas without Wi-Fi. It showcases embedded programming, serial communication, and real-time data handling.

Customized ESP32 - s3 - mini 1

Designed and customized ESP32-S3 Mini-based schematic and PCB layout using Altium Designer. Optimized component placement and routing for reliable performance and compact board design. Ensured proper power management, signal integrity, and manufacturability through DRC and ERC checks.
